DMS Travel

DMS travel is a dynamic application which uses the Facebook API to sign in and retrieve data from the DMS Travel Facebook page. It displays albums from the DMS page that are related to Australia and allows you to access the albums through lightbox. The user is authenticated via a facebook login prompt allowing access to the albums within the application. Each album is displayed with the thumbnail of that specific album on the main page and upon selection will redirect to a page containing all the albums images
